# Сравнение LLM на практической задаче

## 1. Цель задания
Познакомиться с современными языковыми моделями, научиться работать с API и сравнить возможности трех различных LLM на практической задаче, связанной с темами AI Learning Week.

---

## 2. Задача 3: Генератор технических описаний для AI-проектов
**Выполнил:** Гопиенко Александр  
**Команда:** Экспонента  
**Время:** 2-3 часа  
**Сложность:** ⭐⭐ Низкая-Средняя

### Что нужно сделать
1. Создать 8–10 кратких описаний идей AI-проектов (2–3 предложения каждое).
2. Попросить LLM сгенерировать для каждого:
   - Развернутое техническое описание
   - Список необходимых технологий/библиотек
   - Основные этапы реализации (3–5 пунктов)
   - Оценку сложности (легко/средне/сложно)
3. Сравнить качество генерации у трех моделей

**Пример входа:**  
*"Система автоматического обнаружения лесных пожаров с помощью дронов и компьютерного зрения"*

**Критерии сравнения:**
- Техническая грамотность
- Полнота описания
- Практичность рекомендаций

---
