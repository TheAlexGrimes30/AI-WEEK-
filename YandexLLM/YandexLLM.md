# YandexGPT: Генерация AI-проектов

## 1. Краткое описание модели

**YandexGPT** — это LLM (Large Language Model) от Яндекса, предназначенная для генерации текстов, идей и технических описаний.  
Модель способна:  
- Писать развернутые описания проектов;  
- Создавать списки необходимых технологий и библиотек;  
- Предлагать этапы реализации и оценку сложности;  
- Работать через API с ключом авторизации.

**Идея:** Система рекомендаций для образовательных платформ, которая анализирует поведение пользователей и предлагает им наиболее подходящие курсы и материалы.

- Pandas, NumPy
- NLTK, spaCy
- Apache Spark
- MongoDB, PostgreSQL
- Django, Flask
- Matplotlib, Seaborn

**Этапы реализации:**
1. Сбор и предобработка данных пользователей.
2. Разработка и обучение модели машинного обучения.
3. Интеграция модели с платформой.
4. Тестирование и оптимизация системы.
